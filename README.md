# TSP-team project

Поскольку классический алгоритм Lin–Kernighan предназначен для задачи коммивояжёра, в которой требуется посетить все вершины, он не может напрямую применяться к задаче персонализированного туристического маршрута с ограничениями по времени и бюджету. Поэтому мною был использован двухэтапный подход: на первом этапе формируется допустимое подмножество достопримечательностей (с учетом выполнения всех ограничений по цене и времени), а на втором этапе порядок их обхода оптимизируется с помощью алгоритма Lin–Kernighan.
